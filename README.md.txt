# 🚀 Leave Management System

## 📌 Overview
A full-stack role-based leave management system.

## ⚙️ Tech Stack
- React (Frontend)
- ASP.NET Core (Backend)
- SQL Server
- JWT Authentication
- log4net (Logging)

## 👥 Roles
- Admin → approves users & leaves
- Manager → manages teams & approves leaves
- User → requests leave

## 🔁 Workflow
User → Manager → Admin  
Status: Pending / Approved / Rejected

## 📁 Structure
/frontend → React app  
/backend → .NET API  

## ✅ Features
- Role-based authentication
- Team management
- Leave workflow
- Secure APIs

✅ Developed as a production-style system
